# Dart Fundamentals

A structured, concept-by-concept guide to learning Dart — built as a teaching resource.

## Who this is for
Beginners learning Dart as a path to Flutter development.

## How to use this repo
Each folder is a topic. Each file inside covers a single concept with runnable examples.

## Prerequisites
- [Dart SDK installed](https://dart.dev/get-dart)
- VS Code with the [Dart extension](https://marketplace.visualstudio.com/items?itemName=Dart-Code.dart-code)

## Running any file
```bash
dart run 01_variables/01_var_explicit_types.dart
```
